ABOUT ME
I am currently in 2B, studying Software Engineering at the University of Waterloo. I went to Brampton Centennial Secondary School, where I was awarded the Governor General's Academic Medal, which is presented to the student with the highest graduating average.

I developed a passion for programming in Grade 10 when I was introduced to Turing. The simplicity of the language allowed me to look beyond the syntax and experience the essence of code - logic. That year, I developed replicas of a couple of popular games, Pong and Brick Breaker. I have also created versions of games such as Space Invaders and The World's Hardest Game using Java. Then, I went on to work with a team to develop an original Android App, BubblEscape which has ~500 downloads on the Play Store.

Not only do I have experience in creating games, I am also an avid Web Developer. I have worked in a team to develop WhiteBoard, winner of the Best Design Award at HackWaterloo. It is an online platform for developers to connect. I also led a team to develop TransitCheck, a community-driven solution to today's transit problems. Over the years, I have gained experience in Java EE, C, C++, Node.js, PHP, HTML, CSS, JavaScript and various frameworks and libraries that go along with these languages.

SKIP THE WAIT
DESCRIPTION
Skip The Wait is a web application that acts as a check-in system for patients and walk-in clinics. Patients can see the closest clinic near them, the number of patients already in the queue and can queue in before physically going to the clinic. Clinics can see the patients in the queue and the medical reason for which they are coming in. This application helps patients save time and organizes the way walk-in clinics manage patients.
DEVELOPMENT
I developed this app with a team of four students. We made this app in 36 hours at Hack The North. We used Node.js and MongoDB to create the server and Angular.js, HTML and CSS for the client. I created some of the API endpoints for the client to access and used Google Maps API to display the clinics near the patient for them to queue into.


TRANSITCHECK
DESCRIPTION
It is a web application that informs users of the situational delays for TTC. A community driven solution to the day-to-day problems of unexpected delays in transit buses. Users can simply send a message to their followers to notify them of the delay.

DEVELOPMENT
I led the development with a team of four students. We made this app in 24 hours at BattleHack Toronto. We used the Express.js framework supported by Node.js for the backend, Passport.js for authentication and MongoDB as the database. In addition, we used libraries such as Bootstrap and JQuery to design the app.
